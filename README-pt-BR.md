# Calculadora de Calorias

[🇺🇸 English Version](README.md)

App Android para rastrear a ingestão diária de calorias.

## Funcionalidades

- Cadastrar refeições com nome, data, hora, tipo e calorias
- Visualizar refeições por intervalo de datas
- Estatísticas: total de calorias, média diária, calorias por tipo de refeição
- Referência de limite diário: 2100 kcal

## Tipos de refeição

Café da manhã, Desjejum, Almoço, Café da tarde, Festa, Jantar, Sobremesa, Outro

## Requisitos

- Android 8.1+ (API 27)
- Java 8

## Rodando

Abra no Android Studio e execute em um dispositivo ou emulador.

## Stack

- Java
- SQLite (armazenamento local)
- Android SDK 32
