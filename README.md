# Calories Calculator

[Versão em Português](README-pt-BR.md)

Android app to track daily calorie intake.

## Features

- Register meals with name, date, time, type, and calories
- View meals by date range
- Statistics: total calories, daily average, breakdown by meal type
- Daily limit reference: 2100 kcal

## Meal types

Breakfast, After Breakfast, Lunch, Afternoon Snack, Happy Hour, Dinner, Dessert, Other

## Requirements

- Android 8.1+ (API 27)
- Java 8

## Running

Open in Android Studio and run on a device or emulator.

## Stack

- Java
- SQLite (local storage)
- Android SDK 32
